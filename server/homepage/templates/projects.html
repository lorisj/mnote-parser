Projects page:

Below is a list of some of my current projects. If you have an interest or any thoughts for any of these please let me know.

MNotes

MNotes is a math wiki containing notes with formal proofs on math topics such as:

  • Quantum information theory
  • Classical information theory
  • Model Theory
  • Computability Theory

Theorems, definitions and examples are catogorized in a graph database, so if a proof uses a definition/theorem/example, you can easily find all the context you need.

QCCAS

QCCAS is a highly flexible quantum circuit simulator written in python that uses density operators. Rather than using floating point arithmetic, QCCAS uses a computer algebra system (sympy) to give exact outputs, and shows all possible outputs of a quantum circuit. It also allows for n-ary qudits, and other operations such as partial trace and purification.

Structured Coding

Structured coding is a writeup of some ideas I had about generalizing the notion of a (algebraic) structured code using model theory/universal algebra. It also includes a summary of model theory in the appendix.

It is currently in the early stages, once I read up more on group codes and algebraic geometry I will give it some more thought.
