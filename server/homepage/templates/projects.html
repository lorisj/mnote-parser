MNotes

MNotes is a math wiki containing notes with formal proofs on math topics such as:

  • Quantum information theory
  • Classical information theory
  • Model Theory
  • Computability Theory
  • And many more.

Theorems, definitions and examples are catogorized in a graph database, so if a proof uses a definition/theorem/example, you can easily find all the context you need.

QCCAS

QCCAS is a highly flexible quantum circuit simulator written in python that uses density operators. Rather than using floating point arithmetic, QCCAS uses a computer algebra system (sympy) to give exact outputs, and shows all possible outputs of a quantum circuit. QCCAS includes analysis on complexity, and proofs of accuracy for optimization methods used.
